Overview

This Shiny-style dashboard is built with static HTML and GitHub Pages. It provides a home for KID Museum’s school outreach data, interactive maps, and future forecasting charts.

What’s in this dashboard?
Quick tour of the tabs (similar to a Shiny app layout).
  • Maps: Interactive Folium map of schools, programs, and equity buffers.
  • Forecasting: Placeholder for time-series and capacity charts.
  • School Demographics: Placeholder for tables/visuals on Title I, CEP, FARMs, URM, etc.
  • About: Context on the project and data sources.

About This Dashboard

This is a static, Shiny-style data site built with HTML, CSS, and a little JavaScript, hosted on GitHub Pages. It is designed to support KID Museum’s data storytelling and sponsor conversations.

How it works
Technical notes in simple language.
  • The site is hosted on GitHub Pages in the KID-MUSEUM-DATA repository.
  • Tabs at the top are controlled by a small JavaScript function (like Shiny’s tabs).
  • The main map is a Folium HTML export embedded via <iframe>.
  • More charts or maps can be added later as additional HTML files and iframes.
